
BlockPuzzleGame 拼图消除游戏

一、项目简介

这是一个基于 HarmonyOS 和 ArkTS 开发的方块拼图消除游戏。

玩家通过点击预览方块，再点击主网格，将方块合理摆放，当形成 4x4 满格区域时自动消除并获得分数。

二、主要功能

随机生成 10 种不同形状的方块

方块支持顺时针旋转

点击预览方块后再点击网格进行放置

第一次点击显示灰色预览，再次点击确认放置

倒计时自动放置方块

消除 4x4 满格区域并播放动画

支持动态扩展网格大小

实时显示分数

三、游戏操作流程

点击预览区域中的方块格子

点击主网格中的目标位置

再次点击同一位置确认放置

方块形成 4x4 满格时自动消除

倒计时结束时自动放置方块

可随时点击旋转按钮调整方块方向

四、单元格状态说明

空单元格显示为白色

已放置的方块显示为蓝色

预览位置显示为灰色

正在消除的单元格显示为金色动画

五、核心方法说明

initGame 初始化游戏网格和方块

generatePreviewBlock 随机生成方块

rotateBlock 旋转当前方块

canPlaceBlock 判断是否可放置

placeBlockAt 将方块写入网格

checkAndClear 检测并消除满格区域

autoPlaceBlock 倒计时结束后自动放置

expandGrid 扩展网格尺寸

onPreviewBlockClick 处理预览方块点击

onGridClick 处理主网格点击

六、开发环境

开发工具为 DevEco Studio

开发语言为 ArkTS

系统版本为 HarmonyOS API 9 或以上

支持设备为手机和模拟器

七、运行方式

打开 DevEco Studio

导入本项目

连接真机或启动模拟器

点击运行按钮启动游戏

八、注意事项

消除动画播放期间不可操作

页面退出时会自动清理定时器

预览状态不会影响真实网格数据

支持热重载实时更新界面

九、适用场景

适合 HarmonyOS 初学者学习 ArkTS

适合理解状态管理和 UI 更新机制

适合作为拼图类小游戏参考示例
